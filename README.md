# URL Shortener

URL shortener helps you shorten unwieldy links and share branded links with custom domains at scale.

## Usage

### Shorten

```shell
curl -X POST --location "http://localhost:18080" \
    -d "https://www.github.com/"
```

### Redirect

http://localhost:18080/KGA6Kg
